# Virus Total Url Scanner API
 
## Installation
npm install axios
npm install dotenv

1. Create .env file 
2. Go to Virus Total and create an API key.
3. Paste the API key to .env file: `VIRUSTOTAL_API_KEY = xxx`
4. run the app as node index.js
